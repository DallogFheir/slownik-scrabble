# Słownik Scrabble

Aplikacja mobilna, która umożliwia sprawdzanie słów dozwolonych w grze Scrabble w [Oficjalnym Słowniku Polskiego Scrabblisty](https://www.pfs.org.pl/osps.php).

Aby sprawdzić słowo, wystarczy wpisać je w pole tekstowe, a następnie nacisnąć przycisk "Szukaj". Jeśli słowo istnieje w słowniku, płytka z literą przyjmie kolor zielony, w przeciwnym wypadku kolor czerwony.

Długie naciśnięcie na płytkę otworzy słownik [SJP](https://sjp.pl/) z definicją danego słowa.
